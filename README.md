# Yar-Sale Station (Weather Cat)

Веб-панель мониторингаЯр-Сале, объединяющая погодные условия, статус рейсов и состояние зимников в интерактивном визуальном интерфейсе с анимациями.

## Особенности
*   **Визуализация погоды**: Анимированные солнце, луна, облака, снег и дождь, отражающие текущие условия.
*   **Мониторинг рейсов**: Отображение информации о самолетах и вертолетах (`flights.json`).
*   **Статус зимников**: Отслеживание состояния зимних автодорог (`zimnik.json`).
*   **Интерактивность**: Персонаж "Кот в коробке" и различные анимации (фейерверки, гирлянды).
*   **Звуковое сопровождение**: Фоновая музыка и звуки ветра.
*   **Автоматизация**: Скрипты для сбора данных (скрейпинга) с использованием Playwright.

## Технологии
*   HTML5 / CSS3 / JavaScript (Vanilla)
*   Node.js & Playwright (для сбора данных)
*   GitHub Actions (для автоматического обновления данных)

## Установка

1.  Клонируйте репозиторий:
    ```bash
    git clone <repository-url>
    ```
2.  Установите зависимости (для скриптов обновления данных):
    ```bash
    npm install
    ```

## Использование

### Запуск веб-интерфейса
Просто откройте файл `index.html` в любом современном веб-браузере.

### Обновление данных
Для ручного обновления данных о погоде и статусах запустите скрипт скрейпинга:

```bash
npm run scrape
```
Это запустит `node scripts/scrape.js` (и другие связанные скрипты), который обновит файлы JSON и CSV.

## Структура проекта
*   `assets/` — изображения и ресурсы.
*   `css/` — стили интерфейса и анимаций.
*   `js/` — логика приложения (погода, анимации, отрисовка).
*   `scripts/` — Node.js скрипты для сбора данных.
*   `sound/` — аудиофайлы (фоновая музыка, звуки).
*   `.github/workflows/` — конфигурация автоматического обновления данных.
